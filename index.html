Angular Compatibility Prototype

Test shared components across Angular versions

Angular 15 Host

Zone.js change detection, no signals, standalone components

Angular 19 Host (v20 Simulation)

Signals API, modern patterns, standalone components

Angular 19 Host - ZONELESS

No Zone.js - demonstrates CD collision issue with async Web Component events

About This Prototype

Both hosts consume the same shared Web Components from /shared. This tests the pattern of extracting v14 monolith code into framework-agnostic components that can be consumed by newer Angular apps.
