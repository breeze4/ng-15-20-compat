# Host v15

Angular 15 dashboard application - main entry point at `/`.

## Purpose

Tests that shared components built for v15 work in an Angular 15 host with zone.js change detection.

## Routes

- `/overview` - Dashboard overview
- `/analytics` - Analytics view
- `/reports` - Reports view

## Commands

```bash
# From workspace root
pnpm serve:host    # Dev server on port 4200
pnpm build:host    # Production build
```
